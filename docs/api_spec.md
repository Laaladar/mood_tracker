# Спецификация API (Mood Tracker)

Взаимодействие между клиентом и сервером осуществляется по протоколу HTTP с использованием формата JSON.

## 1. Аутентификация

### Регистрация пользователя
**Эндпоинт:** `POST /api/register`

**Тело запроса (JSON):**
```json
{
    "login": "user123",
    "nickname": "Ринат",
    "email": "user@example.com",
    "password": "mypassword"
}
```
**Ответы:**
- `201 Created`: Пользователь успешно создан и авторизован.
- `400 Bad Request`: Логин уже занят.

---

### Вход в систему
**Эндпоинт:** `POST /api/login`

**Тело запроса (JSON):**
```json
{
    "login": "user123",
    "password": "mypassword"
}
```
**Ответы:**
- `200 OK`: Успешный вход. Возвращает объект пользователя.
- `401 Unauthorized`: Неверный логин или пароль.

---

### Выход из системы
**Эндпоинт:** `GET /api/logout`

**Описание:** Очищает серверную сессию пользователя.

---

## 2. Дневник настроения

### Получение записей (History)
**Эндпоинт:** `GET /api/mood`

**Требования:** Требуется активная сессия пользователя.

**Ответ (JSON):**
```json
{
    "nickname": "Ринат",
    "logs": [
        {
            "id": 1,
            "score": 8,
            "word": "Радость",
            "why": "Запустил деплой на Render",
            "happy": "Да",
            "date": "08.05.2026, 14:00"
        }
    ]
}
```

---

### Создание записи
**Эндпоинт:** `POST /api/mood`

**Тело запроса (JSON):**
```json
{
    "score": "8",
    "word": "Радость",
    "why": "Всё работает!",
    "happy": "Да",
    "date": "08.05.2026, 14:00"
}
```
**Ответ:**
- `201 Created`: Запись успешно сохранена в БД.
- `401 Unauthorized`: Пользователь не авторизован.
